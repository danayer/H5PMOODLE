## Введение в H5P

H5P (HTML5 Package) — это открытая и бесплатная технология для создания интерактивного контента. Она позволяет преподавателям создавать разнообразные интерактивные элементы обучения, которые можно встраивать в систему управления обучением (LMS), такую как Moodle.

Основные преимущества H5P:
- Интерактивность и вовлечение
- Множество типов контента (более 40 различных видов активностей)
- Адаптивность для мобильных устройств
- Возможность повторного использования
- Поддержка мультимедиа

## Что такое Drag the Words

Drag the Words (Перетаскивание слов) — это тип активности в H5P, который позволяет создавать упражнения, где учащимся необходимо перетаскивать слова в правильные места в тексте. Этот формат особенно полезен для:

- Изучения языков
- Формирования правильного порядка слов
- Заполнения пропусков в тексте
- Проверки знания терминологии
- Улучшения понимания грамматических структур

## Установка H5P в Moodle

Прежде чем создавать активности Drag the Words, необходимо установить H5P в вашей системе Moodle:

1. Войдите в Moodle как администратор
2. Перейдите в раздел "Администрирование сайта" > "Плагины" > "Установка плагинов"
3. Нажмите кнопку "Установить плагины из каталога плагинов Moodle"
4. Найдите "H5P" в списке плагинов и установите его
5. Следуйте инструкциям по установке
6. После завершения установки активируйте плагин

## Создание активности Drag the Words

### Шаг 1: Добавление активности H5P в курс

1. Перейдите на страницу вашего курса в Moodle
2. Включите режим редактирования, нажав кнопку "Режим редактирования" в правом верхнем углу
3. В нужном разделе курса нажмите "Добавить элемент или ресурс"
4. В открывшемся окне выберите "H5P интерактивный контент" и нажмите "Добавить"

### Шаг 2: Настройка основных параметров

1. Введите название для вашей активности в поле "Название"
2. При необходимости добавьте описание в поле "Описание"
3. В разделе "Настройки H5P" нажмите кнопку "Создать" или "Загрузить контент"
4. Из списка доступных типов контента выберите "Drag the Words"

### Шаг 3: Создание содержимого Drag the Words

1. В поле "Задание" введите краткое описание задания для студентов
2. В поле "Текст задания" введите текст с отмеченными словами для перетаскивания

Ключевой момент: Чтобы обозначить слово или фразу для перетаскивания, используйте специальный синтаксис — заключите слово в звездочки (*). Например:

```
Moodle - это *система* управления *обучением*, которая используется для создания *онлайн-курсов*.
```

В этом примере слова "система", "обучением" и "онлайн-курсов" будут доступны для перетаскивания.

### Расширенный синтаксис Drag the Words

#### Добавление подсказок

Вы можете добавить текстовую подсказку для перетаскиваемого слова, используя двоеточие (:) после слова внутри звездочек:

```
Столица Франции - это *Париж:город на реке Сена*.
```

В этом примере слово "Париж" нужно перетащить в пропуск, а текст "город на реке Сена" будет отображаться как подсказка при наведении курсора на пустое место.

#### Важные особенности перетаскиваемых слов

- Для каждого пустого места есть только одно правильное слово или фраза.
- Слова будут отображаться в случайном порядке в области перетаскивания.
- По умолчанию, все перетаскиваемые слова находятся в общем "банке слов", откуда их можно перетаскивать в любое место.

#### Добавление отзыва для конкретных слов

Вы можете добавить отзыв, который будет отображаться после завершения задачи для конкретных слов:

- Используйте `\+` в конце слова (внутри звездочек) для отзыва на правильный ответ
- Используйте `\-` в конце слова (внутри звездочек) для отзыва на неправильный ответ

Пример:
```
Солнце встает на *востоке\+Правильно! Солнце всегда встает на востоке.*. 
Солнце садится на *западе\-Неверно. Солнце садится на западе, а не на востоке.*.
```

В этом примере, если учащийся правильно перетащит слово "востоке", он увидит отзыв "Правильно! Солнце всегда встает на востоке." Если же учащийся неправильно разместит слово, он увидит соответствующий отрицательный отзыв.

## Подробные настройки Drag the Words

### Текст задания и форматирование

- **Задание**: Введите здесь общие инструкции для учащихся
- **Текст задания**: Основной текст с отмеченными словами для перетаскивания
  - Используйте звездочки (*) для обозначения перетаскиваемых слов
  - Если вам нужно использовать звездочку как обычный символ, используйте обратный слеш перед ней: \*

### Настройки обратной связи и оценивания

Помимо индивидуальных отзывов для конкретных слов, H5P позволяет настроить общую обратную связь:

- **Общий отзыв**: Отображается для всех учащихся независимо от результата
- **Отзыв на основе результата**: Настраиваемые сообщения для разных уровней выполнения
  - Например: "Отлично!" для 90-100%, "Хорошая работа!" для 70-89%, "Попробуйте еще раз" для 0-69%
- **Баллы**: Можно настроить вес каждого правильного ответа в общей оценке задания
- **Попытки**: Настройка количества допустимых попыток

### Дополнительные настройки

- **Обратная связь**: Настройте персонализированные сообщения для разных уровней достижений
  - "Для любого результата" - сообщение для любого результата
  - "От 0% до 100%" - настройте разные сообщения для разных процентных диапазонов
  
- **Поведение**:
  - **Мгновенная обратная связь**: Если активировано, студенты получают мгновенный отклик на свои действия
  - **Проверить все ответы**: Добавляет кнопку для проверки всех ответов сразу
  - **Пробовать снова**: Разрешает студентам повторные попытки
  - **Показать решение**: Позволяет студентам увидеть правильные ответы
  - **Чувствительность к регистру**: Если включено, "Слово" и "слово" будут считаться разными ответами
  - **Требовать ввода всех правильных ответов перед показом решения**: Студентам необходимо попытаться ответить на все вопросы

- **Внешний вид**:
  - **Включить полноэкранный режим**: Добавляет кнопку для расширения активности на весь экран
  - **Показывать заголовок**: Отображает название задания в активности
  - **Выровнять перетаскиваемые элементы по вертикали**: Влияет на расположение перетаскиваемых слов

### Шаг 4: Сохранение и публикация

1. После завершения всех настроек нажмите "Сохранить и показать"
2. Проверьте, как работает ваша активность с точки зрения студента
3. При необходимости вернитесь к редактированию и внесите изменения

## Примеры упражнений по авиационному английскому

### Пример 1: Основная терминология авиационной логистики с подсказками

```
*Air cargo:перевозка грузов по воздуху* is the transportation of goods via an *aircraft:летательный аппарат*. Air cargo can be transported commercially or privately in cargo *aircraft\+Correct! Aircraft is the right term for planes and other flying vehicles.* or in the *payload:груз* section of passenger aircraft. Many *airlines:авиакомпании\-Incorrect. Airlines refers to companies operating aircraft.* derive a significant portion of their revenue from air cargo transportation.
```

```
The air logistics process begins with *cargo acceptance\+Correct! This is the first step in air logistics.*, followed by *weighing\-No, this comes after cargo is accepted.* and *measuring*. Then goods undergo *security screening\+Perfect! Security is paramount in air transport.* before being loaded into *Unit Load Devices:специальные контейнеры*. These containers are then transported to the *aircraft* and secured in the *cargo hold\+Yes! This is where cargo is stored during flight.* for *transportation*.
```

The primary document in air cargo is the *Air Waybill* (AWB). It serves as a *contract* between the shipper and the *carrier*. Other important documents include the *Customs Declaration*, *Dangerous Goods Declaration* (if applicable), and *Certificate of Origin* for international *shipments*.

## Советы и лучшие практики

1. **Оптимальная длина текста**: Старайтесь не делать тексты слишком длинными, чтобы упражнение оставалось наглядным и удобным для выполнения.

2. **Количество перетаскиваемых слов**: Рекомендуется делать от 5 до 15 перетаскиваемых слов в одном упражнении. Слишком малое количество делает задание слишком простым, а слишком большое — утомительным.

3. **Сложность слов**: Подбирайте слова для перетаскивания в соответствии с уровнем знаний учащихся.

4. **Контекст**: Убедитесь, что контекст текста достаточен, чтобы студенты могли определить правильное место для каждого слова.

5. **Обратная связь**: Настройте информативную обратную связь, которая поможет студентам учиться на своих ошибках.

6. **Предварительный просмотр**: Всегда проверяйте, как работает ваше упражнение с точки зрения студента, прежде чем публиковать его.

7. **Разнообразие**: Комбинируйте Drag the Words с другими типами активностей H5P для создания разнообразной учебной среды.

8. **Технические термины**: При работе с техническими текстами (например, по авиационному английскому) сначала убедитесь, что студенты знакомы с терминологией.

9. **Использование подсказок**: Добавляйте подсказки для сложных или специфических терминов, особенно для начинающих учащихся.

10. **Индивидуальные отзывы**: Используйте специфические отзывы для наиболее важных терминов, чтобы усилить обучающий эффект.

11. **Постепенное усложнение**: Создавайте серию упражнений с постепенным увеличением сложности - от базовых терминов с подсказками до сложных предложений без подсказок.

12. **Комбинированное использование**: Сочетайте в одном упражнении слова с подсказками и без них, с обратной связью и без неё, в зависимости от важности понятий.
